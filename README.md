# Enquiry Tracker

Simple full-stack enquiry tracking web application.

## Tech Stack

### Frontend

* React
* Vite
* Styled Components
* Redux Toolkit
* React Router
* Axios
* React Icons

### Backend

* Node.js
* Express.js

### Database

* PostgreSQL
* Prisma ORM

---

## Project Structure

```text
01_Enquiries/
├── frontend/
├── backend/
```

---

## Goals

This project is being built primarily for learning and portfolio purposes.

Main focus areas:

* modern React architecture
* frontend/backend communication
* REST APIs
* Prisma + PostgreSQL
* authentication patterns
* reusable design systems
* deployment workflow

---

## Features (planned)

* Create enquiries
* View enquiries
* Update enquiry status
* Delete enquiries
* Responsive UI
* Admin section
* API integration
* Database persistence

---

## Development

Frontend runs using Vite development server.

Backend will run separately using Node.js + Express.

Both applications will later be connected through API requests.
