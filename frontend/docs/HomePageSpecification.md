# Frontend Specification

## General Style

- Modern clean corporate design
- Large typography
- Rounded corners on cards and buttons
- Light gray backgrounds for content sections
- Dark header and footer
- Purple/indigo accent color
- Spacious layout with large paddings and gaps
- Smooth hover animations and transitions
- Responsive layout for desktop, tablet and mobile

---

# Header

## Desktop Header

### Layout
- Logo on the left
- Navigation menu on the right
- Horizontal alignment
- Full-width dark background
- Sticky/fixed on scroll (optional later)

### Navigation Items
- Home
- Services
- About
- FAQ
- Contact

### Desktop Dropdown Behaviour
- Dropdown appears on hover
- Smooth fade/slide animation
- Dark dropdown background
- White text
- Rounded corners
- Small spacing between dropdown items

---

## Mobile Header

### Layout
- Logo on the left
- Hamburger menu icon on the right

### Mobile Navigation Behaviour
- Full-screen overlay menu OR slide-in side menu
- Dark background
- Vertical navigation items
- Large touch-friendly spacing
- Close icon at top right
- Smooth slide/fade animation
- Menu closes on:
  - link click
  - outside click
  - close button

---

# Hero Section

## Layout
- Full-width background image
- Dark/purple overlay
- Large headline text
- Small review/rating area below text

## Content
- Large marketing headline
- Small supporting text
- Review avatars
- Star rating
- Small trust message

## Behaviour
- Background image covers entire section
- Text vertically centered
- Responsive font scaling

---

# Features Cards Section

## Layout
- 3 cards in desktop view
- 1 card per row on mobile
- Cards slightly overlap previous section

## Card Design
- Rounded corners
- White/light background
- Large icon
- Heading
- Short description text

## Behaviour
- Hover lift animation
- Soft shadow on hover

---

# About Section

## Layout
- Two-column desktop layout
- Image on left
- Text content on right
- Single-column on mobile

## Content
- Large heading
- Description paragraph
- Read more button/link

## Behaviour
- Responsive stacking on mobile
- Image maintains aspect ratio

---

# FAQ Section

## Layout
- FAQ accordion on left
- Illustration/image on right
- Mobile layout stacks vertically

## Accordion Behaviour
- Click to expand/collapse
- Only one item open at a time (optional)
- Smooth open/close animation
- Plus/minus icon animation

## FAQ Item Design
- Rounded borders
- Large readable text
- Light background

---

# Footer

## Layout
- Dark full-width background
- Multiple columns

## Footer Columns
### Column 1
- Logo
- Short company description

### Column 2
- Useful links

### Column 3
- Additional links

### Column 4
- Contact information

## Footer Behaviour
- Stack columns vertically on mobile
- Center align content on small screens

---

# Global Responsive Rules

## Desktop
- Max container width
- Large paddings and gaps
- Multi-column layouts

## Tablet
- Reduced paddings
- Smaller typography
- Some sections stack vertically

## Mobile
- Single-column layouts
- Hamburger navigation
- Reduced spacing
- Large touch targets
- Full-width cards/buttons

---

# Animation Guidelines

## General
- Smooth transitions
- Duration around 0.2s–0.4s
- Ease or cubic-bezier timing

## Hover Effects
- Slight scale or lift
- Shadow increase
- Color transition

---

# Accessibility

- Proper semantic HTML
- Keyboard accessible navigation
- Visible focus states
- Sufficient color contrast
- Alt text for images

---

# Future Expansion Ideas

- Dark/light mode
- Language switcher
- Authentication pages
- Admin dashboard
- Contact form backend integration
- CMS integration
